### `fft.fft`
Returns the 2D Fourier transform of `x`.


#### Args:
- **x** (None): The array to be transformed.
- **expansion** (None): The field expansion to be used.
- **axes** (None): The axes to be transformed, with default being `(-2, -1)`, the final axes.

#### Returns:
- **None**: The transformed `x`.
